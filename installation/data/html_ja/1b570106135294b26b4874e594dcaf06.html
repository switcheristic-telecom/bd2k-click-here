レトロゲームで遊ぼう特集 　　　　　　　→[本文へ]
最近はちょっとしたレトロゲームブームだというのをご存知でした？　Java 機能搭載の携帯電話の登場で、ちょっと昔のゲームを携帯電話で楽しめるサービスを各社スタート。パソコンでも復刻版が次々に発売されています。今回の特集では、一世を風靡したレトロゲームでパソコン用に復刻されたものをピックアップ。確かに最新ゲームは面白いのですが、レトロゲームにはレトロゲームなりの味があって意外とハマってしまうのです。

ACDSee 4.0J 　　　　　　　　　　　　　　→[本文へ]
デジタルカメラで撮った写真やインターネットから手に入れた画像が一杯になって整理に困って いたり、見たい画像ファイルがなかなか見つからなかったりすることはありませんか。こんな画像ファイルの整理に威力を発揮するのが、高速かつ 多機能な画像ビューア ・ 画像管理ソフト「ACDSee 4.0J」です。 ファイル名の一括変換 ・ 幅広いファイルの種類への対応・圧縮ファイル内の画像表示などWindows XPにもない機能を簡単に使うことができます。

XPturbo 　　　　　　　　　　　　　　　　→[本文へ]
WindowsXP、最新の Windowsとして使いやすい OSと言えますが、やはり使っているうちに「もっと速くならないかな」とか「この表示は変更したい」という不満が出てくるものです。 こうした不満の大部分は Windowsをチューニングすることで改善されるのをご存じでしょうか。使い方や使用環境に合わせて Windowsを最適化すれば、使い勝手もぐっとアップするはずです。 「XPturbo」は、WindowsXPを最適化してくれるソフトです。高度な設定項目も、ウィザード機能で簡単に設定することが可能です。細かなカスタマイズもできますし、万が一の場合にはインストール時に近い状態にも戻せますので安心です。
